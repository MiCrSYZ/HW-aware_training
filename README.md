# HW-Aware Training

This repository is organized by research branch.

The currently public code is on the `memristor-extract` branch:

- GitHub: [memristor-extract](https://github.com/MiCrSYZ/HW-aware_training/tree/memristor-extract)
- Local checkout:

```bash
git fetch origin
git switch memristor-extract
```

## Available Branches

| Branch | Status | Contents |
|--------|--------|----------|
| `memristor-extract` | Public | Memristor-aware neural network training and evaluation, including device non-idealities, weight-to-conductance mapping, and hardware-aware training. |
| `main` | Index only | Minimal landing page for the repository. |

## Notes

The `main` branch intentionally does not contain the experiment code. Use the
research branch above for installation instructions, configuration examples,
tests, and runnable training scripts.
